# The-methodology-of-taking-values-into-account-in-the-development-strategy-of-the-city
Разработка методики учёта ценностей населения в стратегии социально-экономического развития города (на примере городов Приволжского федерального округа)
Работа посвящена разработке методики учета ценностей населения в стратегии социально-экономического развития города с пространственной привязкой на уровне вернакулярных районов. Методика включает проведение интервью и опросов, анализ социальных сетей, формирование ценностной карты и матрицы согласования стратегических инициатив. Апробация на примере Уфы, Казани и Салавата показала возможность выявления локальных ценностных профилей и их интеграции в стратегическое планирование. Предложенный подход повышает адресность управленческих решений, снижает риски их отторжения и способствует более эффективной реализации стратегии социально-экономического развития города.
